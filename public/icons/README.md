# PWA Icons

请在此目录放置以下尺寸的图标文件：

- `icon-72x72.png` - 72x72 像素
- `icon-96x96.png` - 96x96 像素
- `icon-128x128.png` - 128x128 像素
- `icon-144x144.png` - 144x144 像素
- `icon-152x152.png` - 152x152 像素
- `icon-192x192.png` - 192x192 像素（必需，用于安装提示）
- `icon-384x384.png` - 384x384 像素
- `icon-512x512.png` - 512x512 像素（必需，用于启动画面）

## 图标设计建议

1. **背景色**：使用与主题色一致的颜色（#1e293b 或 #0f172a）
2. **图标内容**：建议使用相机或视频相关的简洁图标
3. **格式**：PNG，支持透明背景
4. **风格**：简洁扁平化设计，在深色背景上清晰可见

## 生成工具

可以使用以下工具生成：
- [PWA Asset Generator](https://pwa-asset-generator.nicepkg.cn/)
- [Favicon.io](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
